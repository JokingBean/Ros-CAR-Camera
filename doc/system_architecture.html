二、摄像头捕获链路

2.1 V4L2 硬件参数预置

OpenCV 的 CAP_PROP_AUTO_EXPOSURE 在 V4L2 上映射错误（设为 1 实际是手动模式），因此用 v4l2-ctl 直接写硬件寄存器：

参数值说明
auto_exposure1 (Manual)锁定曝光，消除帧间亮度波动
exposure_time_absolute60固定曝光时间
white_balance_automatic1自动白平衡（避免固定色温偏色）
contrast42对比度
sharpness48锐度
gain30限制增益上限

三台相机使用完全相同的参数，消除相机间亮度/色差差异。

2.2 ROI 裁剪加速

利用 EKF 预测的下一帧位置，将全图 2560×1440 裁剪为 300×300 ROI 进行 Tag 检测，大幅降低计算量。仅在连续丢失 >4 帧时才回退全图检测。

ROI_BASE  = 300   # 初始 ROI 边长（像素）
ROI_EXPAND = 200  # 丢失时每次扩大
ROI_MAX_MISS = 4  # 连续丢失次数阈值

三、Tag 检测与位姿解算

3.1 检测流程

CLAHE 增强
clip=2.5, 8×8 → pupil-apriltags
Detector → solvePnP
Tag→世界坐标

3.2 质量过滤（三级过滤）

在 _process_detections() 中对每个检测到的 Tag 做三级过滤，从源头剔除低质量观测：

过滤项阈值作用
像素对角长度> 40px过滤远距离小 Tag，信噪比低
重投影误差< 1.5px过滤 solvePnP 解算质量差的 Tag
平面倾斜角< 40°过滤过于倾斜的 Tag，透视变形大

3.3 立方体中心模型

Tag 贴在 25cm 立方体各面中心，Tag 中心到立方体中心偏移 12.5cm。利用 Tag 航向角将偏移旋转到世界坐标系并修正：

CUBE_HALF = 0.125  # 立方体半边长 (m)
TAG_SIZE  = 0.134  # Tag 边长 (m)

# 将 Tag 中心换算成立方体中心
cube_x = tag_x - CUBE_HALF × cos(yaw)
cube_y = tag_y - CUBE_HALF × sin(yaw)

这样不同面上 Tag 的检测结果都投影到同一个立方体中心，消除因不同 Tag 暴露导致的偏差。

四、多相机融合算法

4.1 融合流程

按相机分组
同相机多Tag取中位数 → 全局中位数
参考点 → 高斯软加权
sigma=4cm → 8cm截断
脏观测降权 → GSD加权
平均

4.2 核心逻辑（fuse_positions）

  1. margin 过滤：优先 ≥30，不足时降至 ≥20
  2. 按相机分组：同一相机内多个 Tag 取中位数（使用 cube_xy 立方体中心坐标）
  3. 全局中位数参考：所有相机位姿的中位数作为参考点
  4. 高斯软加权：偏离中位数越远的相机权重越低，sigma = 4cm
  5. 偏差截断：偏离 >8cm 的相机权重 ×0.01，避免脏观测拉扯融合结果
  6. GSD 加权：结合 GSD（地面采样距离，越近权重越高）
  7. 输出 n_obs：全局有效 Tag 总数（非相机数），供 EKF 自适应

4.3 融合效果示意

场景旧方案（二值剔除）新方案（软加权+截断）
相机A偏离 7cm被剔除 → obs 从3变2 → 位置跳变权重降至 0.29 → 微幅影响 → 位置渐变
相机A偏离 12cm被剔除 → obs 跳变权重 ×0.01 → 几乎忽略 → 位置稳定
obs 计数相机数量（1~3）全局 Tag 总数（如实反映观测冗余）

五、EKF 状态估计器

5.1 模型

参数值含义
状态向量[x, y, yaw, v]位置、航向、速度
过程噪声 Qdiag(0.008, 0.008, 0.02, 0.04)模型不确定性
观测噪声 Rdiag(0.05, 0.05, 0.10)测量不确定性（基准值）
运动模型恒速x += v·cos(yaw)·dt

5.2 自适应观测噪声

根据全局有效 Tag 数 n_obs 分档调节：

n_obsR_scaleEKF 行为滑动窗口权重
≥3（多 Tag 冗余充足）×1.0信任测量，快速响应当前 0.7 / 历史 0.3
=2（中等冗余）×1.8适度平滑当前 0.5 / 历史 0.5
=1（仅 1 个 Tag，约束最弱）×5.0强烈平滑，依赖预测当前 0.3 / 历史 0.7

5.3 航向异常检测

观测航向与预测航向差 > 90° 时，只更新 XY 位置、跳过 yaw 更新，防止航向跳变污染位置估计。

6.1 硬件层

  • 已实施 三台相机锁定完全一致的 V4L2 参数（曝光/对比度/锐度/增益）

6.3 融合层

  • 已实施 高斯软加权 + 8cm 偏差截断
  • 已实施 obs 按全局有效 Tag 数统计
  • 已实施 滑动窗口 4 帧加权平滑

6.4 滤波层

  • 已实施 EKF 自适应观测噪声分档
  • 已实施 航向异常检测（>90°跳过 yaw 更新）
  • 已实施 静止先验（Q XY 分量 0.008）

6.5 标定层

  • 待优化 多相机联合标定：三台相机同时拍摄全场 Tag，统一求解外参
  • 待优化 标定后锁定外参，运行中禁止动态重标定
