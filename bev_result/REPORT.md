## 三、俯视图融合

| 参数 | 值 |
|------|-----|
| 范围 | X: 0.0 ~ 4.5m, Y: 0.0 ~ 5.0m |
| 分辨率 | 200 px/m (总 1000×1100 px) |
| 地面 Tag | 110 个 (50cm 间距网格) |
| 有效覆盖率 | 67.9% |
| 重叠区域 | 两相机在 Y≈2-3m 区域交汇 |

融合方法：
1. 对每个地面像素 (x, y, z=0) 用外参反投影到各相机图像坐标
2. 采样对应像素颜色，构建各相机独立俯视图
3. 重叠区域取两相机像素平均值融合
4. 叠加 110 个地面 Tag 位置和相机位置标注

## 五、后续建议

1. **USB 外参优化**: 在 Pi 上跑 `python calibrate_extrinsics.py --camera usb --mode apriltag` 重标一次，预期误差降到 < 5px
2. **实时 BEV**: 将 `bev_fusion.py` 的投影逻辑整合到 `main.py` 的追踪循环中
3. **立方体追踪验证**: 放置贴有 Tag 的目标立方体，验证多相机融合定位精度
4. **USB 分辨率**: 树莓派上可尝试 2560×1440 MJPG（V4L2 已确认支持），标定后用更高分辨率提升精度
